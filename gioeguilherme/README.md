# Site Gio & Guilherme

Site simples e elegante para casamento, feito com Next.js + Tailwind CSS e pronto para deploy na Vercel.

## Como usar

1. Instale as dependências:

```bash
npm install
```

2. Rode localmente:

```bash
npm run dev
```

3. Build para produção:

```bash
npm run build
npm start
```

Depois é só subir este projeto para um repositório no GitHub e importar na Vercel.
